# 📚 المراجع العلمية للمشروع

## الكتاب المرجعي
- **`فيزياء_1_المنهج_السعودي.pdf`** — الكتاب المدرسي الأصلي (PDF، 214 صفحة).
- **`فيزياء_1_المنهج_السعودي.txt`** — النص الكامل المستخرج من الكتاب (1.6MB).

## كيفية الاستخدام
هذه الملفات **مرجع علمي فقط** — لا تُجمَّع ضمن التطبيق ولا تُحمَّل في المتصفح.
عند بناء أو تعديل أي درس في `src/data/curriculum.ts` أو إضافة أسئلة في
`src/data/quizzes.ts`، يجب الرجوع إلى هذا الكتاب لضمان:

1. **توافق المسميات** مع الكتاب الرسمي.
2. **دقة المحتوى العلمي** والمعادلات الفيزيائية.
3. **اقتباس الأسئلة** من تقويم الفصل في الكتاب.
4. **استخدام أمثلة عددية** من الكتاب نفسه عند الإمكان.

## ملاحظات تقنية
النص العربي قد يحتوي علامات تشكيل (Unicode combining marks) بسبب طبيعة
الـ PDF الأصلي. عند البحث، استخدم كلمات قصيرة أو رقم الصفحة بدلاً من
الجمل الكاملة.
